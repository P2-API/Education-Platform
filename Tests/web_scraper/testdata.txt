        <div class="bottom-column__why-cookies"><h2>Hvad er en cookie?</h2>
<p>En cookie er en lille datafil, som bliver gemt på din computer, tablet eller mobiltelefon. En cookie er ikke et program, der kan indeholde skadelige programmer eller virus.</p> 
<h2>Hjemmesidens brug af cookies</h2>
<p>Cookies kan være nødvendige for at få hjemmesiden til at fungere. Cookies hjælper os desuden med at få overblik over dit besøg på hjemmesiden, så vi løbende kan optimere og målrette hjemmesiden til dine behov og interesser. Cookies husker f.eks., hvad du har lagt i en eventuel indkøbskurv, om du tidligere har besøgt siden, om du er logget ind og hvilket sprog og valuta, du gerne vil have vist på hjemmesiden. Vi bruger også cookies til at målrette vores annoncer over for dig på andre hjemmesider. Helt overordnet anvender vi cookies som del af vores service til at vise indhold, der er så relevant som muligt for dig.</p>
<p>Du kan se, hvilke services der sætter cookies, og til hvilke formål under de forskellige kategorier [Nødvendige, Funktionelle, Statistiske og Marketing].</p>
<h2>Hvor længe bliver cookies gemt?</h2>
<p>Det varierer, hvor længe en bestemt cookie bliver opbevaret på dine enheder og browsere. En cookies levetid beregnes ud fra dit seneste besøg på hjemmesiden. Når cookiens levetid udløber, bliver den automatisk slettet. Alle vores cookies' levetid fremgår i vores cookiepolitik.</p>
<h2>Sådan afviser eller sletter du cookies</h2>
<p>Du kan altid afvise alle -/tredjepartscookies fuldstændig ved at ændre indstillingerne i din browser på din computer, tablet eller telefon. Hvor du finder indstillingerne afhænger af, hvilken browser du anvender. Du skal dog være opmærksom på, at hvis du afviser alle -/tredjepartscookies, vil der være funktioner og services, du ikke kan anvende på hjemmesiden (fordi disse er afhængige af cookies).<br>
</p>
<h2>Hvordan sletter man cookies?</h2>

Hvilken enhed/browser bruger du - klik på det relevante link:</p>

function trapFocusInModal(modal){
  var defaultTemp = document.getElementById(modal);
  var focusableEl = defaultTemp.querySelectorAll('button:not([disabled]), a[href]:not([disabled]), input:not([disabled]), select, textarea:not([disabled]), [tabindex]:not([tabindex="-1"])');
  wrapper.focus();  
  var firstFocusableEl = focusableEl[0];
  var lastFocusableEl = focusableEl[focusableEl.length - 1];
  var KEYCODE_TAB = 9;

  function ctlFocus(e){
    var isTabPressed = (e.key === 'Tab' || e.keyCode === KEYCODE_TAB);
    if (!isTabPressed) { 
      return; 
    }
    if ( e.shiftKey ) /* shift + tab */ {
      if (document.activeElement === firstFocusableEl) {
        lastFocusableEl.focus();
        e.preventDefault();
      }
    } else /* tab */ {
      if (document.activeElement === lastFocusableEl) {
        firstFocusableEl.focus();              
        e.preventDefault();
      }
    }
    document.activeElement.scrollIntoView();
  }
  if (window.addEventListener) {
    defaultTemp.addEventListener('keydown', ctlFocus);
  }else if (document.attachEvent) { // IE 
    defaultTemp.attachEvent("onkeydown", ctlFocus);
  }else{
    defaultTemp.addEventListener('keydown', ctlFocus);
  }
}

        <i class="fa pull-left ug3-dimmed-chevron fa-chevron-down ug3-smooth-transition ug3-upside-down"></i>
                <a id="Indhold" class="anchor"></a>
          <a tabindex="-1" href="#Indhold" id="link-for-anchor-Indhold"><span>Indhold</span></a>      </h2><span class="ug3-foldable-header-sr-label sr-only"> – skjul indhold</span></div>
    
            <div class="views-field-field-bodytext ug3-foldable-content visible-force" style="overflow: hidden; display: block;">
        <div class="field-content"><p>Datalogi betyder egentlig videnskaben om data. Uddannelsen handler ikke kun om computere, men også om de generelle principper bag repræsentation og behandling af data. Og i udviklingen af nye it-systemer har man netop brug for at arbejde systematisk med disse principper.</p><p>Uddannelsens teoretiske fag giver dig den videnskabelige baggrund for at kunne lave modeller af løsningerne på en konkret problemstilling.</p><h3>Undervisningens indhold</h3><p>I løbet af uddannelsen har du kurser som:</p><ul><li>Programmering, herunder de forskellige sprog,&nbsp;som bruges til programmering eller udvikling af nye systemer</li><li>Softwarearkitektur, herunder softwares opbygning og funktion i forhold til andre dele af et it-system</li><li>Databaser, herunder opbygning og udvikling</li><li>Algoritmer og datastrukturer, herunder analyse og konstruktion af&nbsp;de logiske systemer, der ligger bag alt software og fx gør informationer tilgængelige og søgbare</li><li>Systemudvikling, herunder udvikling og vurdering af it-systemer i forhold til deres anvendelsesområde</li></ul><p>Du lærer altså, hvad en computer kan bruges til, hvordan man programmerer den, og hvordan software- og it-systemer designes og udvikles. Alt sammen værdifuld viden, når du skal arbejde med computere, programmer og data.&nbsp;</p><h3>Opbygning og indhold kan variere</h3><p>Uddannelsen kan variere i opbygning og indhold afhængigt af uddannelsessted. Læs mere på det enkelte uddannelsessteds hjemmeside.</p><h3>Titel</h3><p>Du får titlen bachelor (BSc) i datalogi. På Roskilde Universitet afhænger titlen af dit individuelle uddannelsesforløb.</p></div>

Kravene gælder optagelse i både kvote 1 og kvote 2.</p><h3>Generelle adgangskrav</h3><ul><li><a href="/videregaaendeuddannelse/altomoptagelse/gymnasiale-eksamener">En&nbsp;gymnasial eksamen&nbsp;</a>(bemærk: krav om overbygning for hf og eux 1. del)</li></ul><p>Læs mere om adgangsgivende eksamener og uddannelser:<br><a href="/videregaaendeuddannelse/altomoptagelse/adgangskrav-til-de-videregaaende-uddannelser">Adgangskrav til de videregående uddannelser</a></p><h3><strong>Specifikke adgangskrav</strong>&nbsp;</h3><ul><li>Dansk A</li><li>Engelsk B</li><li>Matematik A</li></ul><p>Læs om supplering, hvis du mangler fag eller niveauer:<br><a href="/uddannelser/gymnasialeuddannelser/enkeltfagogsupplering/gymnasial-supplering-gs">Gymnasiale suppleringskurser (GSK)</a></p><h3><strong>Lokale adgangskrav</strong></h3><p>Ud over de generelle og specifikke adgangskrav er der lokale adgangskrav. De gælder kun for det enkelte uddannelsessted.</p><h4><br><strong>Københavns Universitet</strong></h4><ul><li>Hvis du søger om optagelse&nbsp;i kvote 1, skal karaktergennemsnittet fra din&nbsp;gymnasiale eksamen være mindst 6,0</li><li>Hvis du søger om optagelse&nbsp;i kvote 2, skal du bl.a. deltage i en kvote 2-prøve. Se mere på uddannelsesstedets hjemmeside</li></ul><h4><br>Aarhus Universitet</h4><ul><li>Hvis du søger i kvote 1, skal karaktergennemsnittet fra din gymnasiale eksamen være mindst 6,0, og du skal have mindst 6,0 i gennemsnit i matematik A.</li><li>​Hvis du søger i kvote 2, skal du bl.a. deltage i en optagelsesprøve. Se mere på uddannelsesstedet hjemmeside.</li></ul><h4><br><strong>Syddansk Universitet</strong></h4><ul><li>Hvis du søger om optagelse&nbsp;i kvote 1, skal karaktergennemsnittet fra den gymnasiale eksamen være på mindst 6,0 eller 7,0 i gennemsnit i matematik A</li><li>Fra 2025 kan du kun søge i kvote 1, hvis du har et karaktergennemsnit fra din gymnasiale eksamen på mindst 6,0</li><li>Hvis du søger om optagelse&nbsp;i kvote 2, skal du deltage i en optagelsesprøve</li></ul><p><a href="http://www.sdu.dk/da/uddannelse/bachelor/kvote_1_og_2/testbaseret_optag_kvote2" target="_blank">Læs mere om optagelsesprøven på uddannelsesstedets&nbsp;hjemmeside</a></p><h3><br><strong>Særskilte adgangskrav for Roskilde Universitet</strong></h3><p>Datalogi og Computer Science (engelsksproget)&nbsp;er bachelorfag, der læses i kombination med et andet bachelorfag på RUC.</p><p>For at læse datalogi og et andet fag eller Computer Science og et andet fag skal du opfylde adgangskravene og søge optagelse på en&nbsp;af følgende bacheloruddannelser:</p><ul><li><a href="/uddannelser/bachelorogkandidatuddannelser/bacheloruddannelser/humanistiskebacheloruddannelser/oevrigehumaniora/humanistisk-bachelor">Humanistisk bachelor</a></li><li><a href="/Uddannelse.aspx?article_id=udb2-humhumtekbas">Humanistisk-teknologiske bachelor</a></li><li><a href="/Uddannelse.aspx?article_id=udb2-natvidbasisudd">Naturvidenskabelig bachelor</a></li><li><a href="https://www.ug.dk/Uddannelse.aspx?article_id=udb2-samfsamfvidbasisudd">Samfundsvidenskabelig bachelor</a></li><li><a href="https://www.ug.dk/uddannelser/bachelorogkandidatuddannelser/bacheloruddannelser/naturvidenskabeligebacheloruddannelser/oevrigenaturvidenskab/international-bachelor-natural-sciences">International Bachelor in Natural Sciences</a></li></ul><h4><br>Kombinationsmuligheder</h4><p>Der er forskellige kombinationsmuligheder for datalogi og Computer Science. Undersøg mulighederne på RUCs hjemmeside:&nbsp;</p><ul><li><a href="http://ruc.dk/bachelor/kombinationer/3127" target="_blank">Datalogi</a>&nbsp;</li><li><a href="http://ruc.dk/bachelor/kombinationer/129" target="_blank">Computer Science</a></li></ul></div>
